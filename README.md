# Random Quote Generator

## 📌 Description
The Random Quote Generator is a simple web application developed as part of the CodeAlpha App Development Internship. It displays inspirational and motivational quotes randomly with a single click.

## ✨ Features
- Displays a random quote on page load
- "New Quote" button to generate another quote
- Displays quote text and author
- Clean and responsive user interface
- Interactive user experience

## 🛠 Technologies Used
- HTML5
- CSS3
- JavaScript

## 🚀 How to Run
1. Download or clone this repository.
2. Open the project folder.
3. Double-click `index.html` or open it in any web browser.
